# Documentation

# Backend

Darant šį projektą nors ir jau buvau dirbes su ASP.NET core Web API, išmokau nemažai.

Išmokau kaip sukūrti tvarkingą autentifikaciją ir kaip pagal REST principus sudaryti routes.

# Frontend

Frontend išmokau, kaip geriau struktūrizuoti kodą, kad apklikacija butu SPA ir nedarytų nereikalingų užklausų.

# Deployment

Išmokau kaip paleisti backend į SolidOcean su docker, taip pat kaip paleist NextJS projektą naudojant Vercel.
