# Register

Registers new user

# URL

GET /api/register

# Parameters

username required

email required

password required

# Example Request Body

{
    "userName":"username",
    "email":"user@example.com",
    "password":"!Password123"
}

# Example Response

{
  "userId": 0,
  "username": "username",
  "email": "user@example.com";
}

# Response codes

  • 400 - request was invalid or api failed to create user.
  • 201 - user was created
